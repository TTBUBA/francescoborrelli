🎨 Sistema di Animazioni Avanzate

Pubblicato il 3 Giugno 2026 | Lettura: 6 minuti

Introduzione

Gestire animazioni complesse può essere complicato, ma con il giusto sistema diventa facile. Impariamo come usare l'Animator Controller per creare transizioni fluide.

1. Setup dell'Animator

Crea un Animator Controller e aggiungi i parametri:


using UnityEngine;

public class AnimationController : MonoBehaviour
{
    private Animator animator;
    
    void Start()
    {
        animator = GetComponent<Animator>();
    }

    public void PlayIdle()
    {
        animator.SetTrigger("Idle");
    }

    public void PlayRun()
    {
        animator.SetBool("IsRunning", true);
    }

    public void StopRun()
    {
        animator.SetBool("IsRunning", false);
    }

    public void PlayAttack()
    {
        animator.SetTrigger("Attack");
    }
}

2. Transizioni Fluide

Crea transizioni smooze tra animazioni:


// Nel Player Movement Script
void HandleAnimation()
{
    Vector3 move = new Vector3(Input.GetAxis("Horizontal"), 0, Input.GetAxis("Vertical"));
    
    if (move.magnitude > 0)
    {
        animator.SetFloat("Speed", move.magnitude, 0.1f, Time.deltaTime);
    }
    else
    {
        animator.SetFloat("Speed", 0, 0.1f, Time.deltaTime);
    }
}

Usa Blend Trees per interpolazioni smooth:

5. Synchronizzare Animazioni Complesse

Coordina più animatori per combo attacks:


public class ComboSystem : MonoBehaviour
{
    private Animator animator;
    private int comboCount = 0;
    private float comboResetTime = 2f;
    private float lastAttackTime = 0f;

    void Update()
    {
        if (Time.time - lastAttackTime > comboResetTime)
        {
            comboCount = 0;
        }

        if (Input.GetMouseButtonDown(0))
        {
            PerformCombo();
        }
    }

    void PerformCombo()
    {
        comboCount++;
        lastAttackTime = Time.time;
        
        if (comboCount == 1)
            animator.SetTrigger("Attack1");
        else if (comboCount == 2)
            animator.SetTrigger("Attack2");
        else if (comboCount >= 3)
            animator.SetTrigger("Attack3");
    }
}

Best Practices

  • ✓ Usa Animator Parameters invece di Play("NomeAnimazione")
  • ✓ Crea layer separati per movimenti e azioni
  • ✓ Imposta Transition Duration a 0 per reazioni istantanee
  • ✓ Usa Exit Time per transizioni naturali
  • ✓ Testa sempre le animazioni con diverse velocità

Conclusione

Un buon sistema di animazioni rende il gioco fluido e piacevole. Pratica con l'Animator Controller e sperimenta! 🎬
